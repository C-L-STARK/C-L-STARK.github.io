## 程序员疯狂成长之路

- `小白程序员` - 通常，是那些门外汉吧，想要内窥程序员世界的那些人
- `入门程序员` - 搜索引擎、AI 生成、工具、框架、脚本小子
- `初级程序员` - **点** 代码编写能力以及问题解决能力
- `高级程序员` - **线** 有思考的解决某一类问题
- `超级程序员` - **面/体** 有思考的编写面向特定行业解决方案的程序员或者是研究员
- `其他程序员` - 拥有特殊方向特殊技能的有价值群体
